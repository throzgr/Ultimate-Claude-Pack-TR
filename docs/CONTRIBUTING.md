# Contributing

Thank you for contributing to Ultimate Claude Pack TR.

## Contribution Rules

- Follow the project folder structure.
- Document every new Skill.
- Add examples when possible.
- Keep prompts clear and reusable.
- Test your changes before submitting.

## Skill Structure

```
skill-name/
├── README.md
├── examples.md
└── prompt.md
```

## Agent Structure

```
agent-name/
└── README.md
```

## Pull Requests

- Keep changes focused.
- Explain what you changed.
- Update documentation if needed.
